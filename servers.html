Add Your Server

Running a NeoNetrek server? Get it listed here by submitting a pull request to the website repository.

Edit servers.json and add an entry with the following fields:

{
  "name": "Your Server Name",
  "url": "https://your-server-url.example.com",
  "location": "City, Country",
  "region": "region-code",
  "description": "A short description of your server.",
  "admin": "Your Name or Team",
  "adminContact": "https://link-to-contact",
  "established": "2026",
  "features": ["Feature 1", "Feature 2"]
}

Once your PR is reviewed and merged, your server will appear on this page automatically.
